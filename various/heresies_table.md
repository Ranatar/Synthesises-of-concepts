# Таблица христианских ересей по догматическим параметрам

## 1. Количество и природа Божественных Ипостасей

| Позиция | Исторические ереси | Гипотетические ереси |
|---------|-------------------|---------------------|
| **Одна ипостась** | Модализм/савелианство (Савеллий, III в.) - Бог являет себя в трех модусах/масках | *Радикальный монизм* - только Отец существует, Сын и Дух - лишь силы |
| **Три отдельных божества** | Тритеизм (некоторые крайние формы номинализма) | *Олимпийская триада* - три равных независимых бога без единства сущности |
| **Сын не единосущен Отцу** | Арианство (Арий, IV в.) - Сын сотворен; Полуарианство - Сын подобосущен | *Градуированная иерархия* - Дух ниже Сына, Сын ниже Отца, все три - разные степени божественности |
| **Дух не божественен** | Пневматомахи/македониане (IV в.) - Дух - творение или сила | - |
| **Четыре ипостаси** | - | *Тетратеизм* - добавление четвертой ипостаси (например, Софии-Премудрости) |

## 2. Характер взаимоотношений Ипостасей

| Позиция | Исторические ереси | Гипотетические ереси |
|---------|-------------------|---------------------|
| **Филиокве** | Западное учение (с IX в.) - Дух исходит от Отца и Сына | - |
| **Дух от Сына** | - | *Пневматологический субординатизм* - Дух исходит только от Сына, минуя Отца |
| **Сын от Духа** | - | *Обратная процессия* - Сын рождается при участии Духа или от Духа |
| **Взаимное исхождение** | - | *Циклическая триадология* - все три Ипостаси порождают друг друга по кругу |
| **Временное появление Сына** | Динамический монархианизм (Павел Самосатский) - Сын стал Богом во времени | - |

## 3. Природы в Иисусе Христе

| Позиция | Исторические ереси | Гипотетические ереси |
|---------|-------------------|---------------------|
| **Одна природа** | Монофизитство (Евтихий, V в.) - одна божественная природа поглотила человеческую; Аполлинарианство - неполная человеческая природа (без ума/духа) | *Одна человеческая природа* - Христос был только человеком, обоженным благодатью |
| **Призрачная плоть** | Докетизм (гностики, I-II в.) - человечество Христа призрачно | - |
| **Смешение природ** | Монофизитство крайнее - слияние природ в одну сложную | *Третья природа* - две природы слились в совершенно новую, богочеловеческую сущность |
| **Разделение природ/ипостасей** | Несторианство (Несторий, V в.) - две природы и фактически две ипостаси, соединенные морально | - |
| **Три природы** | - | *Тринатуризм* - к божественной и человеческой добавляется ангельская или иная природа |
| **Сменяемые природы** | - | *Христологический динамизм* - Христос попеременно был то Богом, то человеком |

## 4. Воли в Иисусе Христе

| Позиция | Исторические ереси | Гипотетические ереси |
|---------|-------------------|---------------------|
| **Одна воля** | Монофелитство (VII в.) - одна богочеловеческая воля | - |
| **Одна энергия** | Монэнергизм (VII в., предшествовал монофелитству) - одна энергия действования | - |
| **Конфликт воль** | - | *Христологический дуализм* - две воли в постоянном конфликте, без гармонии |
| **Три воли** | - | *Триволитизм* - божественная, разумная человеческая и животная/страстная человеческая воли |
| **Поглощение воли** | Аполлинарианство (частично) - человеческий ум заменен Логосом | - |
| **Две ипостаси** | Несторианство (косвенно) - каждой природе соответствует своя ипостась | *Разделенная личность* - две отдельные личности (Иисус и Сын), действующие совместно |

## 5. Природа Церкви

| Позиция | Исторические ереси | Гипотетические ереси |
|---------|-------------------|---------------------|
| **Зависимость от святости клира** | Донатизм (IV в.) - таинства недействительны у недостойных священников | - |
| **Невидимая церковь** | Протестантские учения (частично) - истинная Церковь невидима, видимые церкви несовершенны | - |
| **Церковь без Духа** | - | *Христомонизм* - Церковь исключительно институт Христа, Дух не имеет конституирующей роли |
| **Дух без Христа** | Некоторые харизматические крайности (XX в.) - приоритет переживания Духа над связью со Христом | *Пневматоцентризм* - Церковь создана Духом независимо от Христа |
| **Множественность церквей** | - | *Христологический плюрализм* - каждая община со Христом - отдельная независимая церковь без общего тела |
| **Церковь = земная организация** | Цезарепапизм - полное подчинение Церкви государству | - |
| **Отрицание институционального аспекта** | Духовное христианство (секты типа духоборов, молокан) | *Анархо-пневматология* - Церковь - чистое действие Духа без всякой структуры и таинств |

## Примечания

**Методология:** Таблица составлена по принципу отклонения от православных догматов, принятых на Вселенских Соборах.

**Гипотетические ереси** отмечены курсивом и представляют собой логически возможные, но исторически не реализовавшиеся отклонения от ортодоксии. Некоторые из них могли существовать в зачаточных формах, но не получили оформления как систематические учения.

**Важно:** Многие исторические ереси представлены в упрощенной форме. В реальности богословские позиции были сложнее, и современные исследователи иногда пересматривают классические оценки (например, относительно Нестория или восточных нехалкидонитов).
