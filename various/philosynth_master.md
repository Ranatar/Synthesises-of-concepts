## Профили комбинаций

### ★★★ Структурный резонанс — безусловные рекомендации

---

#### Трансформативный × Диалектический
**Единственная пара с ★★★ во всех шести разделах без исключений.**

Общая логика: оба параметра разделяют один онтологический жест — *продуктивный конфликт как источник нового*. «Напряжение» уровня = пара тезис↔антитезис метода. «Рождение нового из невозможности» = снятие. Два языка для одного процесса.

| Раздел | Механизм резонанса |
|--------|-------------------|
| Резюме | «Фундаментальная несовместимость» (уровень) = «тезис↔антитезис» (метод). «Новый вопрос из несовместимости» = «характер снятия». Один вопрос в двух терминологиях. |
| Граф | Столбец «Генеалогия» (напряжение → разрешение) и диалектическая маркировка таблицы связей производят единый контент: исторический (откуда) + логический (как). |
| Глоссарий | «Трансформированные термины» = термины-синтезы (снятие с предшественниками). «Эмерджентные термины» = снятие без прямых предшественников. Подразделы уровня = диалектические роли метода. |
| Тезисы | «Конфликт породил тезис» (уровень) = «антитезис + снятие» (метод). Шкала эмерджентности и шкала диалектичности — взаимно верифицирующие метрики. |
| Диалог | Шаги 3–5 структуры уровня (несовместимость → рождение понятия → осознание нейтральности) взаимно однозначно совпадают с тремя моментами диалектической процедуры метода. |
| Критика | «Скрытое заимствование» (уровень) = «отсутствие реального антитезиса» (метод). Таблица диалектичности производит данные для проверки эмерджентности автоматически. |

**Когда выбирать:** когда важна прозрачная архитектура синтеза; когда философы образуют явные оппозиции; когда результат должен читаться и как исторически обоснованный, и как логически необходимый.

---

#### Трансформативный × Деконструктивный
**★★★ в пяти разделах из шести (★★ в резюме — единственное отступление).**

Общая логика: «напряжение между системами» (уровень) возникает именно там, где их *скрытые предпосылки несовместимы* (метод). Трансформация = деконструкция + пересборка. Два языка для одного процесса.

| Раздел | Механизм резонанса |
|--------|-------------------|
| Резюме | Уровень запрашивает напряжения; метод спрашивает об их подлинности («не воспроизведёт ли пересборка предпосылки?»). Рефлексия в резюме делает весь синтез самокритичным. (★★) |
| Граф | Столбец «Генеалогия» и деконструктивная маркировка категорий описывают одно: до/после момента деконструкции. Кластеры по напряжениям = кластеры «деконструированных» vs «пересобранных». |
| Глоссарий | «Понятия-предшественники» (уровень) = «традиционные понимания, подлежащие деконструкции» (метод). Столбцы таблицы заполняют требование метода автоматически. |
| Тезисы | «Конфликт породил тезис» (уровень) = «предпосылка преодолена» (метод). Самопроверочный вопрос метода («не возврат ли?») = вопрос уровня («действительно эмерджентен?»). |
| Диалог | Шаги 2–3 (демонстрация несовместимостей) = деконструкция предпосылок. Финал обоих — новые основания, освобождённые от разоблачённых допущений. |
| Критика | «Возврат к предпосылке» (метод) = «скрытое заимствование» (уровень). Таблица деконструктивности производит материал для оценки трансформативности. |

**Когда выбирать:** когда важна философская честность по отношению к источникам; когда синтез должен демонстрировать, *почему* исходные системы не могли прийти к результату самостоятельно; когда нужен встроенный механизм самопроверки.

---

#### Порождающий × Деконструктивный
**★★★ во всех шести разделах. Максимальный критический потенциал матрицы.**

Общая логика: оба параметра разделяют один эпистемологический жест — *выявление того, чего не видно изнутри существующих систем*. «Слепое пятно» (уровень) = «нераспознанная предпосылка» (метод). Единственная пара, где обе оси *независимо* требуют доказать, почему ни одна исходная система не могла сформулировать результат.

| Раздел | Механизм резонанса |
|--------|-------------------|
| Резюме | «Что философ не может увидеть» (уровень) = «предпосылки, подлежащие деконструкции» (метод). Слепое пятно = нераспознанная предпосылка. Один вопрос. |
| Граф | 6-й столбец «Преодолённые ограничения» = деконструированная предпосылка + пересборка. Типология «деконструированные / пересобранные» — естественная классификация порождающих категорий. |
| Глоссарий | Трёхчастная структура каждой словарной статьи: (1) что было за скобками, (2) почему структурно невидимо, (3) как термин это преодолевает. Уровень задаёт части 1 и 3, метод — часть 2. |
| Тезисы | «Ограничение системы» (уровень) = «предпосылка, делающая тезис невозможным» (метод). Проверка «не возврат ли?» = «действительно невозможен в рамках традиции?». Наиболее строгий критерий тезисов. |
| Диалог | Шаги 2–3–4–5 уровня (ограничения → коллективная недостаточность → новый аппарат → набросок) = деконструктивная процедура метода (предпосылки → разоблачение → пересборка). Полное совпадение. |
| Критика | «Подлинная пересборка или возврат» (метод) и «убедительна ли невыводимость» (уровень) — одна и та же проверка в двух системах. Двойной аудит одного возможного провала. |

**Когда выбирать:** когда зерно концепции формулирует проблему, которую ни один из философов не мог поставить в рамках своей системы; когда важна максимальная философская строгость в обосновании новизны; для наиболее амбициозных синтезов с высокими претензиями на оригинальность.

---

#### Порождающий × Аналитический
**★★★ в тезисах и критике; ★★ в остальных разделах. Нарастающая синергия.**

Общая логика: порождающий уровень *заявляет* о невыводимости из традиций — аналитический метод предоставляет *формальный аппарат для доказательства* этого заявления. Синергия нарастает к разделам, где нужно верифицировать претензии, а не генерировать контент.

| Раздел | Механизм резонанса |
|--------|-------------------|
| Резюме | «Что философ не может увидеть» + аналитический вопрос «что допускает формализацию?» → формализованное слепое пятно как более сильный исходный материал. (★★) |
| Граф | Доказательство невыводимости через отсутствие genus в таксономии исходных традиций. Аналитические определения превращают утверждения о новизне в верифицируемые тезисы. (★★) |
| Глоссарий | «Почему ни один философ не мог сформулировать термин» (уровень) + строгое определение через genus/differentia (метод) = формальное доказательство невозможности. (★★) |
| Тезисы | Если в цепочке логических шагов нет предпосылок из исходных традиций — тезис верифицированно порождён проблемой. Шкала порождающей генерации перестаёт быть интуитивной. **(★★★)** |
| Диалог | Формальная демонстрация неразрешимости (шаг 2 уровня) через явную цепочку шагов (метод). «Неразрешимость» как доказательство, а не утверждение. (★★) |
| Критика | Аналитический аудит предпосылок определяет оценку по шкале 0–10 (выводимость из одной традиции). Наиболее строгая верификация порождающих претензий. **(★★★)** |

**Когда выбирать:** когда синтез претендует на подлинную новизну и должен это доказать; когда аудитория философски требовательна и ориентирована на аналитическую традицию; когда важна аудируемость результата.

### ★★ Продуктивный парадокс — рекомендации с пояснением

---

#### Трансформативный × Аналитический
**★★ во всех шести разделах. Равномерная стабильная синергия.**

Аналитический метод делает трансформативные претензии *верифицируемыми*: «эмерджентность» перестаёт быть нарративным утверждением и превращается в результат формального аудита предпосылок. Оценка по шкале эмерджентности 0–10 определяется проверкой: если в логической структуре категории или тезиса присутствует предпосылка одной традиции — её трансформативность не полна.

**Когда выбирать:** когда нужна трансформативная глубина с аналитической строгостью; когда важно не просто заявить об эмерджентности, но доказать её структурно.

---

#### Трансформативный × Творческий
**★★ во всех шести разделах. Равномерная продуктивность.**

Трансформативный уровень создаёт категории с неустановленными отношениями — они принципиально новее, чем их связи, которые ещё не кристаллизовались. Разрешение творческого метода маркировать связи как гипотетические здесь не слабость, а *честная онтология нового*. «Генеративные узлы» топологии = наиболее трансформированные категории.

**Когда выбирать:** когда важна свобода в установлении отношений между трансформативными категориями; для синтезов в экспериментальном или проектном контексте.

---

#### Порождающий × Творческий
**★★★ в глоссарии и диалоге; ★★ в остальных. Пики в «открытых» разделах.**

Оба параметра отказываются от закрытого финала: уровень — нащупывает контуры нового языка, метод — производит карту вопросов. Единственная комбинация, где оба параметра *явно* не требуют завершённого синтеза. Таблица диалога (уровень) производит карту вопросов, которую творческий метод требует как финал — совпадение итоговых артефактов.

**Когда выбирать:** для максимально экспериментальных синтезов; когда ценность — в постановке новых вопросов, а не в закрытой системе; когда философское зерно само по себе является открытым вопросом, а не утверждением.

### ★ Надёжное качество — рекомендации без оговорок

---

#### Компаративный × (любой метод)
**★ во всех разделах и со всеми методами.**

Компаративный уровень не предъявляет к категориям требований, конфликтующих ни с одним методом: он работает с позициями философов как данностью и не запрещает никакой процедуры их обработки. Всё взаимодействие уровня с методом определяется выбором метода, а не уровня.

**Внутри компаративного уровня** — не все методы одинаковы по продуктивности: Аналитический и Диалектический дают наиболее структурированные результаты; Творческий — наиболее оригинальные; Герменевтический — наиболее контекстуально богатые.

**Когда выбирать компаративный уровень:** когда приоритет — достоверность представления философов, а не новизна; для вводных и образовательных документов; когда аудитория незнакома с синтезируемыми традициями.

### ≈ Управляемое напряжение — с оговорками

---

#### Порождающий × Диалектический
**≈ во всех шести разделах. Единственная некомпаративная пара без жёсткого конфликта, но и без синергии.**

Напряжение управляемо при одном условии: «ограничение философа A» должно интерпретироваться как «тезис», «ограничение философа B» — как «антитезис», «порождённая категория» — как «снятие». Это нетривиальная переинтерпретация диалектических ролей, которую промпт не предписывает явно — модель может уйти в сторону.

**Когда использовать:** только если явно добавить в промпт инструкцию вида «интерпретируй "ограничения" традиций как диалектические полюса, а "порождённую категорию" — как снятие». Без этой инструкции результат непредсказуем.

---

#### Трансформативный × Герменевтический
**≈ в пяти разделах; ✗ в диалоге.**

Ключевая проблема сосредоточена в диалоге: трансформативный уровень строит на несовместимости (несовместимости → рождение нового), герменевтический метод — на расширении горизонтов (совместимости как нарастающей). Диалог получает два несовместимых финала. В остальных разделах напряжение мягкое.

**Когда использовать:** при исключении диалога из состава разделов — напряжение снижается до управляемого ≈. При включении диалога — заменить метод на Деконструктивный или Диалектический.

---

#### Порождающий × Герменевтический
**✗ в четырёх разделах из шести (граф, глоссарий, диалог, критика); ≈ в двух.**

Системный конфликт: герменевтический метод организует весь граф вокруг горизонтов традиций, тогда как порождающий уровень запрещает строить категории из традиций. Четыре раздела получают несовместимые критерии организации содержания.

**Рекомендация:** избегать. Если герменевтический метод принципиален — снизить уровень до компаративного или трансформативного.

---

### ✗✗ Жёсткие конфликты — к замене параметров

---

#### Трансформативный × Интегративный
**✗✗ в пяти разделах из шести; ≈ только в глоссарии.**

Системный конфликт по всем ключевым измерениям. Трансформативный уровень строится на несовместимости как источнике; интегративный метод строится на общих основаниях как источнике. Это взаимоисключающие онтологии синтеза.

| Раздел | Суть конфликта |
|--------|----------------|
| Резюме | «Фундаментальные несовместимости» (уровень) vs «почему философы способны к интеграции» (метод) |
| Граф | «Новое из невозможности совместить» (уровень) vs «ссылка на общее основание ≥ 2 философов» (метод) |
| Тезисы | «Конфликт породил тезис» (уровень) vs «общее основание, из которого тезис вырастает» (метод) |
| Диалог | «Обнаружение несовместимостей → рождение из невозможности» (уровень) vs «кооперативный поиск согласия» (метод) |
| Критика | Метод оценивает «общие основания»; уровень проверяет «напряжения». Взаимно неприменимые системы оценки. |

**Замена для сохранения уровня:** Диалектический или Деконструктивный.
**Замена для сохранения метода:** Компаративный.

---

#### Порождающий × Интегративный
**✗✗ в пяти разделах; ✗ в глоссарии.**

Максимальный конфликт матрицы. Порождающий уровень строит категории из проблемы, не из традиций; интегративный метод требует строить категории из общих оснований традиций. Взаимоисключение тотально.

| Раздел | Суть конфликта |
|--------|----------------|
| Резюме | «Что философ не может увидеть» (уровень) vs «общие основания традиций» (метод) |
| Граф | «Категории из проблемы, не из систем» (уровень) vs «ядро пересечений традиций» (метод) |
| Тезисы | «Тезис невозможен в рамках философии» (уровень) vs «тезис вырастает из общего основания» (метод) |
| Диалог | «Коллективная несостоятельность систем» (уровень) vs «кооперативный поиск согласия» (метод) |
| Критика | Метод проверяет «общие основания»; уровень доказывает их принципиальное отсутствие. |

**Замена для сохранения уровня:** Деконструктивный (★★★), Аналитический (★★★ в тезисах и критике), Творческий (★★).
**Замена для сохранения метода:** Компаративный (★).
**Нет совместимой замены одного параметра при ★★+ результате:** параметры фундаментально противоположны по онтологии.

## Итоговые рекомендации для системы параметров

### По уровням синтеза

| Уровень | Рекомендуемые методы | Допустимые с оговорками | Несовместимые |
|---------|---------------------|------------------------|---------------|
| Компаративный | Все (★ со всеми) | — | — |
| Трансформативный | Д ★★★, Дк ★★★, А ★★, Тв ★★ | Г (исключить диалог) | И |
| Порождающий | Дк ★★★, А ★★★, Тв ★★ | Д (добавить инструкцию), Г (только без графа/диалога/критики) | И |

### По методам синтеза

| Метод | Рекомендуемые уровни | Допустимые с оговорками | Несовместимые |
|-------|---------------------|------------------------|---------------|
| Диалектический | Тр ★★★, К ★ | П (добавить инструкцию) | — |
| Интегративный | К ★ | — | Тр, П |
| Деконструктивный | Тр ★★★, П ★★★, К ★ | — | — |
| Герменевтический | К ★ | Тр (без диалога) | П |
| Аналитический | П ★★★, Тр ★★, К ★ | — | — |
| Творческий | П ★★, Тр ★★, К ★ | — | — |

### Матрица замен при конфликте

| Конфликтная пара | Сохранить уровень → заменить метод | Сохранить метод → заменить уровень |
|-----------------|------------------------------------|------------------------------------|
| Тр × И | → Д (★★★) или Дк (★★★) | → К (★) |
| П × И | → Дк (★★★) или А (★★★) | → К (★) |
| П × Г | → Дк (★★★) или А (★★) | → К (★) или Тр (≈) |
| Тр × Г | → Д (★★★) или Дк (★★★) при включённом диалоге | → К (★) |

### Ключевые наблюдения

**Деконструктивный — единственный метод без конфликтов на любом уровне**, и единственный, достигающий ★★★ одновременно с трансформативным и порождающим уровнями. Его центральный жест (выявление скрытых предпосылок) изоморфен «напряжению» трансформативного уровня и «ограничению» порождающего.

**Интегративный — единственный метод с жёсткими конфликтами на некомпаративных уровнях** и единственный, не образующий ★★+ ни с каким некомпаративным уровнем. Его онтология синтеза (из общего) диаметрально противоположна онтологии трансформативного (из несовместимого) и порождающего (из проблемы).

**Герменевтический работает только на компаративном уровне**. На трансформативном — умеренный конфликт в диалоге; на порождающем — системный конфликт в четырёх разделах. Метод семантически привязан к идее «горизонтов традиций», что несовместимо с любым уровнем, запрещающим организовывать синтез через традиции.

**Диалог — наиболее чувствительный раздел**: именно в нём Тр×Г опускается с ≈ до ✗, а П×Тв и П×Дк достигают ★★★. При выборе проблемных комбинаций диалог следует либо исключать из состава разделов, либо учитывать при выборе параметров в первую очередь.

**Глоссарий — наименее чувствительный раздел**: Тр×И даёт лишь ≈ (а не ✗✗, как везде), П×Тв достигает ★★★ (а не ★★, как везде). Глоссарий «поглощает» конфликты формально и усиливает синергии через открытость структуры словарной статьи.
